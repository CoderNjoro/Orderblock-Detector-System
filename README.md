Orderblock Detector System

Orderblock Detector System is a web application for traders and analysts to identify order blocks and liquidity lines in financial charts. The app lets you enter your Twelve Data API key, select a currency pair, choose a timeframe, and set the number of bars to analyze. With one click, you can view detected order blocks and liquidity lines on a chart in a new tab.

Features

- Order Block Detection: Finds order blocks in your selected market and timeframe.
- Liquidity Line Drawing: Draws key liquidity lines to show potential support and resistance.
- Customizable Analysis: Choose your currency, timeframe, and number of bars.
- Interactive Charts: View clean, interactive financial charts in a new browser tab.
- Secure API Integration: Enter your Twelve Data API key to fetch market data.

How It Works

1. Enter Your API Key
   Input your Twelve Data API key to enable data access.

2. Select Currency Pair
   Choose the currency pair you want to analyze (e.g., EUR/USD, BTC/USD).

3. Select Timeframe
   Pick a timeframe (e.g., 1min, 5min, 1h, 1d).

4. Set Number of Bars
   Specify the number of candlesticks to fetch.

5. Analyze Order Blocks
   Click the "Analyze Order Blocks" button to begin processing.

6. View Results
   A new tab opens showing the chart with highlighted order blocks and liquidity lines.

Installation

1. Clone the Repository
   git clone https://github.com/CoderNjoro/Orderblock-Detector-System.git
   cd Orderblock-Detector-System

2. Install Dependencies
   Ensure you have Python 3.x and pip installed. Then:
   pip install -r requirements.txt

3. Set Up API Key
   You can set your Twelve Data API key as an environment variable or provide it through the app.

Usage

1. Run the Application
   python app.py

2. Open the Web Interface
   Visit http://127.0.0.1:5000 (or your configured port).

3. Fill in the Input Fields
   - Enter your API key.
   - Select currency.
   - Choose timeframe.
   - Set number of bars.
   - Click "Analyze Order Blocks".

4. View Your Analysis
   A chart with detected order blocks and liquidity lines appears in a new tab.

API Information

This app uses Twelve Data API for real-time and historical market data.
